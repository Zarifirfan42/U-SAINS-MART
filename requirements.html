USM Marketplace Requirements

Feel free to share this page or publish it on GitHub to brief collaborators. Key highlights:

  • Multi-page front-end using HTML, CSS, and vanilla JavaScript.
  • User management: registration, login, roles (Student + Admin), profile, inbox.
  • Marketplace flow: add/edit/delete listings, browse catalog with filters, report listings, admin removal.
  • Buyer tools: product details, contact seller/chat, cart, checkout, payment mocks, rating prompt.
  • Communication: inbox, simple chat, mock notifications.

Primary brand colour #4C1D95, secondary #F3E8FF, fonts: Segoe UI / system.

Suggested GitHub repo name: usm-marketplace. Share the repo link so friends can clone and run by opening index.html.
